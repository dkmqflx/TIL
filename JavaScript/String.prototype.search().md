## String.prototype.search()

- search 메소드는 인자로 전달되는 정규식 또는 문자열과 매칭되는지 확인하고

- 만약 매칭된다면 매칭되는 문자열의 첫번째 index를 반환하고

- 그렇지 않다면 -1을 반환한다

```js
"123".search("1"); // 0

"abce".search("ba"); // -1
```
