첫 FastAPI 앱
동작 흐름 · Swagger · cURL

이론을 손으로 옮길 차례입니다. uv로 환경을 만들고, 첫 엔드포인트를 띄우고, 타입 힌트가 어떻게 자동 문서로 바뀌는지 직접 봅니다.

“

요청은 Uvicorn(서버) → FastAPI(앱) → 경로 함수 순서로 흐르고, 반환값은 자동으로 JSON이 되어 되돌아온다. 그리고 타입 힌트는 공짜로 Swagger 문서가 된다.

  • uv로 FastAPI 프로젝트를 세팅하고 서버를 띄운다
  • 요청이 Uvicorn → FastAPI → 경로 함수로 흐르는 과정을 그린다
  • 자동 생성되는 Swagger UI / ReDoc를 활용한다
  • cURL로 터미널에서 API를 테스트한다

1uv로 30초 환경 세팅

uv는 Rust로 만든 초고속 Python 패키지 매니저입니다. 프론트의 npm/pnpm과 거의 같은 멘탈 모델 — pyproject.toml(=package.json)과 락파일로 의존성을 관리합니다.

2첫 엔드포인트

main.py 파일을 만들고 다음을 작성합니다. (전체 예제는 code/main.py에 있습니다.)

main.py
from fastapi import FastAPI

app = FastAPI()

@app.get("/")
def read_root():
    return {"message": "Hello, FastAPI"}

@app.get("/todos/{todo_id}")
def read_todo(todo_id: int, q: str | None = None):
    return {"todo_id": todo_id, "q": q}

서버를 띄웁니다:

terminal
uv run uvicorn main:app --reload

# main  → main.py 파일
# app   → 그 안의 FastAPI() 인스턴스 변수명
# --reload → 코드 바뀌면 자동 재시작 (개발용)
💡 반환값이 알아서 JSON이 된다

return {"message": "..."}처럼 dict를 반환하면 FastAPI가 자동으로 Content-Type: application/json 응답으로 직렬화합니다. Express에서 res.json(...)을 매번 호출하던 것과 달리, 반환만 하면 끝입니다.

주소창에 http://127.0.0.1:8000/todos/42를 치면 내부에서 이런 일이 벌어집니다.

📞 비유 — 호텔 프런트

Uvicorn은 외부 전화를 받아 말을 알아듣는 교환원(ASGI 서버)입니다. FastAPI는 "42번 방 담당이 누구지?"를 찾아 연결하는 지배인(라우팅)이고, read_todo()가 실제 일을 하는 직원입니다. Uvicorn / Starlette / FastAPI가 정확히 어떻게 역할을 나누는지는 Week 4에서 해부합니다.

4타입 힌트의 보상 — 자동 문서

서버를 띄운 채로 브라우저에서 /docs를 열어보세요. 아무 설정도 안 했는데 인터랙티브 API 문서가 떠 있습니다.

주소무엇특징
/docsSwagger UI브라우저에서 직접 요청을 쏴볼 수 있음 (Try it out)
/redocReDoc읽기 좋은 문서형 레이아웃
/openapi.jsonOpenAPI 명세위 문서들의 원천 데이터(JSON)

앞에서 todo_id: int라고 적었기 때문에, 문서에는 "todo_id는 정수"라고 표시되고 /todos/abc처럼 잘못된 요청은 FastAPI가 자동으로 422 검증 에러로 막아줍니다. NestJS에서 Swagger 데코레이터(@ApiProperty)를 일일이 달던 수고가 타입 힌트만으로 사라집니다.

5cURL — 터미널에서 API 두드리기

브라우저 없이 API를 테스트하는 가장 기본 도구가 curl입니다. 백엔드 개발자는 이걸 손에 익혀야 합니다.

terminal
# GET 요청
curl http://127.0.0.1:8000/todos/42

# 쿼리 파라미터 붙이기
curl "http://127.0.0.1:8000/todos/42?q=hello"

# 헤더 보기 (-i: 응답 헤더 포함)
curl -i http://127.0.0.1:8000/

# POST + JSON 본문 (-X 메서드, -H 헤더, -d 데이터)
curl -X POST http://127.0.0.1:8000/todos \
  -H "Content-Type: application/json" \
  -d '{"title": "장보기", "done": false}'
옵션의미
-XHTTP 메서드 지정 (POST, PATCH …)
-H헤더 추가 (보통 Content-Type)
-d요청 본문(body) 데이터
-i응답에 상태줄·헤더까지 함께 출력
⚠️ 자주 하는 실수

-d로 JSON을 보낼 땐 -H "Content-Type: application/json"을 반드시 붙이세요. 빠뜨리면 서버가 폼 데이터로 오해해 422가 납니다. 그리고 URL에 ?·&가 있으면 셸이 해석하지 않도록 전체를 따옴표로 감싸야 합니다.

6✅ 셀프 체크

Q uvicorn main:app에서 main과 app은 각각 뭔가요?

main은 main.py 모듈, app은 그 안에서 app = FastAPI()로 만든 인스턴스 변수 이름입니다. 즉 "main 모듈의 app 객체를 ASGI 앱으로 띄워라"라는 뜻입니다.

Q 설정 한 줄 없이 /docs가 뜨는 이유는?

FastAPI가 경로 함수의 타입 힌트를 읽어 OpenAPI 명세(/openapi.json)를 자동 생성하고, Swagger UI/ReDoc가 그 명세를 렌더링하기 때문입니다. 타입 힌트가 곧 문서의 원천입니다.

Q /todos/abc로 요청하면 어떻게 되나요?

todo_id: int 힌트 때문에 FastAPI가 "abc"를 정수로 변환하지 못하고 422 Unprocessable Entity 응답을 자동으로 돌려줍니다. 함수 본문은 실행조차 되지 않습니다.
