1ConfigModule вҖ” .env мқҪкё°

NestJS м•ұм—җм„ң нҸ¬нҠё, DB URL, JWT мӢңнҒ¬лҰҝ к°ҷмқҖ к°’мқҖ мҪ”л“ңм—җ н•ҳл“ңмҪ”л”©н•ҳл©ҙ м•Ҳ лҗ©лӢҲлӢӨ. @nestjs/configмқҳ ConfigModuleмқ„ лЈЁнҠё лӘЁл“Ҳм—җ н•ң лІҲ л“ұлЎқн•ҳл©ҙ м–ҙлҠҗ м„ңл№„мҠӨм—җм„ңлӮҳ ConfigServiceлЎң кәјлӮј мҲҳ мһҲмҠөлӢҲлӢӨ.

app.module.ts вҖ” ConfigModule м „м—ӯ л“ұлЎқ
import { Module } from '@nestjs/common';
import { ConfigModule } from '@nestjs/config';

@Module({
  imports: [
    ConfigModule.forRoot({
      isGlobal: true,   // лӘЁл“  лӘЁл“Ҳм—җм„ң imports м—Ҷмқҙ мӮ¬мҡ© к°ҖлҠҘ
      envFilePath: '.env',
    }),
  ],
})
export class AppModule {}
some.service.ts вҖ” ConfigServiceлЎң к°’ мқҪкё°
import { Injectable } from '@nestjs/common';
import { ConfigService } from '@nestjs/config';

@Injectable()
export class SomeService {
  constructor(private readonly configService: ConfigService) {}

  getPort(): number {
    return this.configService.get<number>('PORT') ?? 3000;
  }
}
рҹ§ӯ .envлҠ” м ҲлҢҖ м»Өл°Ӣн•ҳм§Җ м•ҠлҠ”лӢӨ

.envм—җлҠ” DB л№„л°ҖлІҲнҳё, JWT мӢңнҒ¬лҰҝ, AWS нӮӨ к°ҷмқҖ лҜјк°җ м •ліҙк°Җ лӢҙк№ҒлӢҲлӢӨ. л°ҳл“ңмӢң .gitignoreм—җ м¶”к°Җн•ҳкі , нҢҖмӣҗ кіөмң лҠ” лі„лҸ„ ліҙм•Ҳ мұ„л„җ(Vault, AWS Secrets Manager)мқ„ мқҙмҡ©н•ҳм„ёмҡ”. л ҲнҸ¬м—җ м»Өл°Ӣн•ҳл©ҙ мқҙлҜё мң м¶ңмһ…лӢҲлӢӨ.

2нҢҢмқј м—…лЎңл“ң вҖ” Multer + FileInterceptor

HTTP multipart/form-dataлЎң м „мҶЎлҗң нҢҢмқјмқҖ NestJS лӮҙмһҘ Multer м—°лҸҷмңјлЎң мІҳлҰ¬н•©лӢҲлӢӨ. м»ЁнҠёлЎӨлҹ¬ л©”м„ңл“ңм—җ @UseInterceptors(FileInterceptor('н•„л“ңлӘ…'))мқ„ лӢ¬кі  @UploadedFile()лЎң нҢҢмқј л©”нғҖлҚ°мқҙн„°лҘј л°ӣмҠөлӢҲлӢӨ.

posts.controller.ts вҖ” нҢҢмқј м—…лЎңл“ң м—”л“ңнҸ¬мқёнҠё
import {
  Controller, Post, UseInterceptors, UploadedFile,
} from '@nestjs/common';
import { FileInterceptor } from '@nestjs/platform-express';
import { diskStorage } from 'multer';
import { extname } from 'path';

@Controller('posts')
export class PostsController {

  @Post('image')
  @UseInterceptors(
    FileInterceptor('image', {
      storage: diskStorage({
        destination: './public/images',
        filename: (req, file, cb) => {
          const unique = Date.now() + '-' + Math.round(Math.random() * 1e9);
          cb(null, unique + extname(file.originalname));
        },
      }),
    }),
  )
  uploadImage(@UploadedFile() file: Express.Multer.File) {
    return { filename: file.filename };
  }
}
рҹ’Ў м—¬лҹ¬ нҢҢмқј м—…лЎңл“ң вҖ” FilesInterceptor

н•ң н•„л“ңм—җм„ң м—¬лҹ¬ нҢҢмқјмқ„ л°ӣмңјл Өл©ҙ FilesInterceptor('images', 10)(мөңлҢҖ 10к°ң)лҘј м“°кі  нҢҢлқјлҜён„°лҘј @UploadedFiles()лЎң ліҖкІҪн•©лӢҲлӢӨ. нҢҢмқј мң нҳ• кІҖмҰқмқҖ fileFilter мҳөм…ҳм—җм„ң file.mimetypeмқ„ нҷ•мқён•ҳл©ҙ лҗ©лӢҲлӢӨ.

3м •м Ғ нҢҢмқј м„ңл№ҷ вҖ” ServeStaticModule

Multerк°Җ public/images/м—җ м ҖмһҘн•ң нҢҢмқјмқ„ HTTPлЎң м ңкіөн•ҳл Өл©ҙ @nestjs/serve-staticмқҳ ServeStaticModuleмқ„ л“ұлЎқн•©лӢҲлӢӨ. rootPathлҠ” л””мҠӨнҒ¬ кІҪлЎң, serveRootлҠ” URL м ‘л‘җмӮ¬мһ…лӢҲлӢӨ.

рҹҚ” л№„мң  вҖ” мӮ¬л¬јн•Ёкіј м—ҙлһҢ м°Ҫкө¬

м—…лЎңл“ңлҠ” нҢҢмқјмқ„ мӮ¬л¬јн•Ё(public/)м—җ л„ЈлҠ” кІғ, Static ServingмқҖ м—ҙлһҢ м°Ҫкө¬лҘј м—¬лҠ” кІғмһ…лӢҲлӢӨ. мӮ¬л¬јн•Ё лІҲнҳё(rootPath)лҘј м•Ңл ӨмЈјкі  м°Ҫкө¬ лІҲнҳё(serveRoot)лҘј м •н•ҳл©ҙ, лҲ„кө¬л“  к·ё м°Ҫкө¬ URLлЎң нҢҢмқјмқ„ кәјлӮҙ ліј мҲҳ мһҲмҠөлӢҲлӢӨ. NestJS мҪ”л“ң м—Ҷмқҙ нҢҢмқј мһҗмІҙк°Җ мқ‘лӢөмқҙ лҗ©лӢҲлӢӨ.

app.module.ts вҖ” ServeStaticModule л“ұлЎқ
import { ServeStaticModule } from '@nestjs/serve-static';
import { join } from 'path';

@Module({
  imports: [
    ServeStaticModule.forRoot({
      rootPath: join(__dirname, '..', 'public'),  // л””мҠӨнҒ¬ кІҪлЎң
      serveRoot: '/public',                           // URL м ‘л‘җмӮ¬
    }),
  ],
})
export class AppModule {}

л“ұлЎқн•ҳл©ҙ public/images/abc.jpgлҠ” GET /public/images/abc.jpgлЎң м ‘к·ј к°ҖлҠҘн•©лӢҲлӢӨ. м—…лЎңл“ң мқ‘лӢөм—җм„ң filenameмқ„ л°ҳнҷҳн•ҳл©ҙ нҒҙлқјмқҙм–ёнҠёлҠ” /public/images/{filename}мңјлЎң мқҙлҜём§ҖлҘј н‘ңмӢңн•  мҲҳ мһҲмҠөлӢҲлӢӨ.

4вң… м…Җн”„ мІҙнҒ¬ вҖ” л©ҙм ‘кҙҖмІҳлҹј л¬јм–ҙліҙкё°

Q .env к°’мқ„ NestJS мҪ”л“ңм—җм„ң мқҪлҠ” л°©лІ•мқ„ м„ӨлӘ…н•ҳм„ёмҡ”.

AppModuleм—җ ConfigModule.forRoot({ isGlobal: true })лҘј л“ұлЎқн•©лӢҲлӢӨ. isGlobal: trueлЎң м„Өм •н•ҳл©ҙ лӢӨлҘё лӘЁл“Ҳм—җм„ң ConfigModuleмқ„ importsн•ҳм§Җ м•Ҡм•„лҸ„ лҗ©лӢҲлӢӨ. к°’мқҙ н•„мҡ”н•ң м„ңл№„мҠӨ мғқм„ұмһҗм—җ ConfigServiceлҘј мЈјмһ…н•ҳкі  this.configService.get<нғҖмһ…>('KEY')мңјлЎң мқҪмҠөлӢҲлӢӨ. .env нҢҢмқјм—җ KEY=value нҳ•мӢқмңјлЎң м„ м–ён•ҳл©ҙ лҗ©лӢҲлӢӨ.

Q м—…лЎңл“ңн•ң нҢҢмқјмқ„ URLлЎң ліҙмқҙкІҢ н•ҳлҠ” м„Өм •мқҖ л¬ҙм—Үмқёк°Җмҡ”?

@nestjs/serve-staticмқҳ ServeStaticModule.forRoot({ rootPath, serveRoot })лҘј мӮ¬мҡ©н•©лӢҲлӢӨ. rootPathм—җ л””мҠӨнҒ¬ кІҪлЎң(мҳҲ: join(__dirname, '..', 'public')), serveRootм—җ URL м ‘л‘җмӮ¬(мҳҲ: '/public')лҘј м§Җм •н•©лӢҲлӢӨ. к·ёлҹ¬л©ҙ public/images/abc.jpg нҢҢмқјмқҙ GET /public/images/abc.jpgлЎң м ‘к·ј к°ҖлҠҘн•ҙм§‘лӢҲлӢӨ. NestJS м»ЁнҠёлЎӨлҹ¬ м—Ҷмқҙ нҢҢмқј мһҗмІҙк°Җ мқ‘лӢөмқҙ лҗ©лӢҲлӢӨ.

Q FileInterceptorмҷҖ FilesInterceptorмқҳ м°ЁмқҙлҠ” л¬ҙм—Үмқёк°Җмҡ”?

FileInterceptor('fieldName')мқҖ лӢЁмқј нҢҢмқј н•ҳлӮҳлҘј л°ӣкі  нҢҢлқјлҜён„°м—җ @UploadedFile()мқ„ м”ҒлӢҲлӢӨ. FilesInterceptor('fieldName', maxCount)лҠ” к°ҷмқҖ н•„л“ңм—җм„ң м—¬лҹ¬ нҢҢмқјмқ„ л°ӣкі  нҢҢлқјлҜён„°м—җ @UploadedFiles()лҘј м”ҒлӢҲлӢӨ. л‘җ кІҪмҡ° лӘЁл‘җ @UseInterceptors()лЎң л©”м„ңл“ңм—җ м Ғмҡ©н•©лӢҲлӢӨ.
