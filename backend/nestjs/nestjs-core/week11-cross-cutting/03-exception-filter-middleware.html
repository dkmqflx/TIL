Exception Filter В· Middleware
м—җлҹ¬ мқ‘лӢө нҶөмқјкіј мһ…кө¬ кІҖл¬ёмҶҢ

@Catch(HttpException)мңјлЎң м—җлҹ¬ мқ‘лӢө нҳ•нғңлҘј нҶөмқјн•ҳкі , NestMiddlewareлЎң мҡ”мІӯмқҙ м»ЁнҠёлЎӨлҹ¬м—җ лӢҝкё° м „ кіөнҶө мІҳлҰ¬(лЎңк№…В·нҢҢмӢұ)лҘј мӮҪмһ…н•ңлӢӨ.

"

MiddlewareлҠ” мҡ”мІӯмқҙ л“Өм–ҙмҳӨлҠ” мһ…кө¬ кІҖл¬ёмҶҢ, Exception FilterлҠ” м—җлҹ¬к°Җ лӮҳк°ҖлҠ” м¶ңкө¬ мқ‘кёүмӢӨ. л‘ҳмқҖ лқјмқҙн”„мӮ¬мқҙнҒҙм—җм„ң м •л°ҳлҢҖ м§Җм җм—җ мң„м№ҳн•ҳм§Җл§Ң, н•Ёк»ҳ м“°л©ҙ мҡ”мІӯВ·мқ‘лӢө м „мІҙлҘј мқјкҙҖлҗҳкІҢ м ңм–ҙн•  мҲҳ мһҲлӢӨ.

  • @Catch(HttpException)кіј ExceptionFilter мқён„°нҺҳмқҙмҠӨлЎң м—җлҹ¬ мқ‘лӢө кө¬мЎ°лҘј нҶөмқјн•  мҲҳ мһҲлӢӨ
  • app.useGlobalFilters() лҳҗлҠ” APP_FILTERлЎң м „м—ӯ л“ұлЎқн•ҳлҠ” л‘җ л°©мӢқмқҳ м°ЁмқҙлҘј м•ҲлӢӨ
  • NestMiddleware.use(req, res, next)лҘј кө¬нҳ„н•ҳкі  configure(consumer)лЎң лқјмҡ°нҠём—җ м Ғмҡ©н•  мҲҳ мһҲлӢӨ
  • Middleware, Guard, Interceptor, Exception Filterк°Җ мҡ”мІӯ лқјмқҙн”„мӮ¬мқҙнҒҙм—җм„ң мӢӨн–үлҗҳлҠ” мҲңм„ңлҘј м„ӨлӘ…н•  мҲҳ мһҲлӢӨ

1Exception Filter вҖ” м—җлҹ¬ мқ‘лӢө нҳ•нғң нҶөмқј

NestJS кё°ліё м—җлҹ¬ мқ‘лӢөмқҖ { statusCode, message, error } нҳ•нғңмһ…лӢҲлӢӨ. н•ҳм§Җл§Ң мӢӨл¬ҙм—җм„ңлҠ” нғҖмһ„мҠӨнғ¬н”„В·кІҪлЎң к°ҷмқҖ л””лІ„к·ё м •ліҙлҘј н•Ёк»ҳ л°ҳнҷҳн•ҳкұ°лӮҳ, н”„лЎңм қнҠё м „мІҙм—җм„ң лҸҷмқјн•ң м—җлҹ¬ нҸ¬л§·мқ„ к°•м ңн•ҳкі  мӢ¶мҠөлӢҲлӢӨ. ExceptionFilterлЎң мӣҗн•ҳлҠ” нҳ•нғңлҘј л§Ңл“Ө мҲҳ мһҲмҠөлӢҲлӢӨ.

code/http-exception.filter.ts
import {
  ExceptionFilter, Catch, ArgumentsHost, HttpException,
} from '@nestjs/common';
import { Request, Response } from 'express';

@Catch(HttpException)
export class HttpExceptionFilter implements ExceptionFilter {
  catch(exception: HttpException, host: ArgumentsHost): void {
    const ctx = host.switchToHttp();
    const response = ctx.getResponse<Response>();
    const request = ctx.getRequest<Request>();
    const status = exception.getStatus();
    const exceptionResponse = exception.getResponse();

    const message =
      typeof exceptionResponse === 'string'
        ? exceptionResponse
        : (exceptionResponse as Record<string, unknown>).message ?? exception.message;

    response.status(status).json({
      statusCode: status,
      timestamp: new Date().toISOString(),
      path: request.url,
      message,
    });
  }
}
рҹ’Ў @Catch()м—җ нҒҙлһҳмҠӨлҘј л„Јмңјл©ҙ к·ё нғҖмһ…л§Ң мһЎлҠ”лӢӨ

@Catch(HttpException)мқҖ HttpExceptionкіј к·ё н•ҳмң„ нҒҙлһҳмҠӨ(NotFoundException, BadRequestException л“ұ)лҘј лӘЁл‘җ мһЎмҠөлӢҲлӢӨ. мқёмҲҳлҘј л№„мҡ°л©ҙ(@Catch()) лӘЁл“  мҳҲмҷёлҘј мһЎмҠөлӢҲлӢӨ. DB м—җлҹ¬мІҳлҹј HttpExceptionмқҙ м•„лӢҢ мҳҲмҷёлҸ„ мІҳлҰ¬н•ҳл Өл©ҙ @Catch()лҘј м“°кі  лӮҙл¶Җм—җм„ң нғҖмһ…мқ„ л¶„кё°н•ҳл©ҙ лҗ©лӢҲлӢӨ.

2м „м—ӯ л“ұлЎқ вҖ” useGlobalFilters vs APP_FILTER

Exception FilterлҘј м „м—ӯмңјлЎң л“ұлЎқн•ҳлҠ” л°©лІ•мқҖ л‘җ к°Җм§Җмһ…лӢҲлӢӨ.

л°©лІ• 1 вҖ” main.tsм—җм„ң мҲҳлҸҷ л“ұлЎқ
// main.ts
const app = await NestFactory.create(AppModule);
app.useGlobalFilters(new HttpExceptionFilter());
await app.listen(3000);
л°©лІ• 2 вҖ” APP_FILTER н”„лЎңл°”мқҙлҚ” (DI к°ҖлҠҘ)
// app.module.ts
import { APP_FILTER } from '@nestjs/core';

@Module({
  providers: [
    {
      provide: APP_FILTER,
      useClass: HttpExceptionFilter,
    },
  ],
})
export class AppModule {}
рҹ§ӯ FilterлҠ” лқјмқҙн”„мӮ¬мқҙнҒҙ л§Ҳм§Җл§ү вҖ” InterceptorмҷҖ м—ӯн•  кө¬л¶„

Exception FilterлҠ” мҡ”мІӯ лқјмқҙн”„мӮ¬мқҙнҒҙм—җм„ң к°ҖмһҘ л§Ҳм§Җл§үм—җ мӢӨн–үлҗ©лӢҲлӢӨ. н•ёл“Өлҹ¬лӮҳ Interceptorм—җм„ң лҚҳм§„ мҳҲмҷёк°Җ н•„н„°лЎң нқҳлҹ¬мҳөлӢҲлӢӨ. InterceptorлҠ” н•ёл“Өлҹ¬мқҳ м •мғҒ нқҗлҰ„мқ„ к°Җкіө(tap)н•ҳкұ°лӮҳ м—җлҹ¬лҘј ліҖнҷҳ(catchError)н•  мҲҳ мһҲлҠ” л°ҳл©ҙ, Exception FilterлҠ” мөңмў… м—җлҹ¬ мқ‘лӢө нҳ•нғңлҘј лӢҙлӢ№н•©лӢҲлӢӨ. л‘җ м—ӯн• мқҖ лӢӨлҰ…лӢҲлӢӨ.

3Middleware вҖ” мҡ”мІӯ мһ…кө¬мқҳ кІҖл¬ёмҶҢ

MiddlewareлҠ” мҡ”мІӯмқҙ лқјмҡ°нҢ…лҗҳкё° м „, NestJS нҢҢмқҙн”„лқјмқёмқҳ к°ҖмһҘ м•һ лӢЁкі„м—җм„ң мӢӨн–үлҗ©лӢҲлӢӨ. Express лҜёл“ӨмӣЁм–ҙмҷҖ лҸҷмқјн•ң (req, res, next) мӢңк·ёлӢҲмІҳлҘј мӮ¬мҡ©н•©лӢҲлӢӨ. лҢҖн‘ң мҡ©лҸ„лҠ” лЎңк№…, мҡ”мІӯ нҢҢмӢұ, CORS, мҶҚлҸ„ м ңн•ңмһ…лӢҲлӢӨ.

рҹҚ” л№„мң  вҖ” мһ…кө¬ кІҖл¬ёмҶҢ vs м¶ңкө¬ мқ‘кёүмӢӨ

Middleware = мһ…кө¬ кІҖл¬ёмҶҢ: кұҙл¬јм—җ л“Өм–ҙмҳӨлҠ” лӘЁл“  мӮ¬лһҢмқ„ кІҖмӮ¬н•©лӢҲлӢӨ. нҶөкіјн•ҳл©ҙ м•ҲмңјлЎң л“Өм–ҙк°‘лӢҲлӢӨ. Exception Filter = м¶ңкө¬ мқ‘кёүмӢӨ: м•Ҳм—җм„ң лӢӨміҗм„ң лӮҳмҳӨлҠ” мӮ¬лһҢмқ„ мІҳм№ҳн•©лӢҲлӢӨ. кІҖл¬ёмҶҢлҘј нҶөкіј лӘ» н•ҳл©ҙ мқ‘кёүмӢӨм—җ лҸ„лӢ¬н•  мқјлҸ„ м—Ҷкі , мқ‘кёүмӢӨмқҖ м •мғҒм ҒмңјлЎң лӮҳк°ҖлҠ” мӮ¬лһҢм—җкІҢлҠ” кҙҖм—¬н•ҳм§Җ м•ҠмҠөлӢҲлӢӨ. л‘ҳмқҖ лқјмқҙн”„мӮ¬мқҙнҒҙм—җм„ң м •л°ҳлҢҖ мң„м№ҳм—җ мһҲмҠөлӢҲлӢӨ.

logging.middleware.ts вҖ” мҡ”мІӯ лЎңк№… лҜёл“ӨмӣЁм–ҙ
import { Injectable, NestMiddleware } from '@nestjs/common';
import { Request, Response, NextFunction } from 'express';

@Injectable()
export class LoggingMiddleware implements NestMiddleware {
  use(req: Request, res: Response, next: NextFunction): void {
    console.log(`[${new Date().toISOString()}] ${req.method} ${req.url}`);
    next();   // л°ҳл“ңмӢң next()лҘј нҳём¶ңн•ҙм•ј лӢӨмқҢ лӢЁкі„лЎң м§„н–үлҗңлӢӨ
  }
}

// app.module.ts вҖ” лқјмҡ°нҠём—җ м Ғмҡ©
export class AppModule implements NestModule {
  configure(consumer: MiddlewareConsumer): void {
    consumer
      .apply(LoggingMiddleware)
      .forRoutes('*');  // лӘЁл“  лқјмҡ°нҠём—җ м Ғмҡ©
  }
}

4вң… м…Җн”„ мІҙнҒ¬ вҖ” л©ҙм ‘кҙҖмІҳлҹј л¬јм–ҙліҙкё°

Q Exception FilterмҷҖ Interceptorмқҳ м—җлҹ¬ мІҳлҰ¬ м°ЁмқҙлҘј м„ӨлӘ…н•ҳм„ёмҡ”.

л‘җ к°Җм§Җ кҙҖм җмқҳ м°Ёмқҙк°Җ мһҲмҠөлӢҲлӢӨ. в‘  лӘ©м Ғ: Interceptorмқҳ catchErrorлҠ” м—җлҹ¬лҘј мһЎм•„ лӢӨлҘё ObservableлЎң ліҖнҷҳн•ҳкұ°лӮҳ л¶Җк°Җ мһ‘м—…(rollback л“ұ)мқ„ мҲҳн–үн•ҳкі  м—җлҹ¬лҘј лӢӨмӢң лҚҳм§‘лӢҲлӢӨ. Exception FilterлҠ” мөңмў…м ҒмңјлЎң HTTP мқ‘лӢө нҳ•нғңлҘј кІ°м •н•©лӢҲлӢӨ. в‘Ў мӢӨн–ү мҲңм„ң: Interceptorмқҳ catchErrorк°Җ лЁјм Җ мӢӨн–үлҗҳкі , м—җлҹ¬к°Җ rethrowлҗҳл©ҙ Exception Filterк°Җ л°ӣм•„ мқ‘лӢөн•©лӢҲлӢӨ. нҠёлһңмһӯм…ҳ лЎӨл°ұмқҖ Interceptorм—җм„ң, м—җлҹ¬ нҸ¬л§· нҶөмқјмқҖ Exception Filterм—җм„ң лӢҙлӢ№н•ҳлҠ” кІғмқҙ м—ӯн•  л¶„лҰ¬мқҳ н•өмӢ¬мһ…лӢҲлӢӨ.

Q MiddlewareлҠ” мҡ”мІӯ лқјмқҙн”„мӮ¬мқҙнҒҙмқҳ м–ҙл””м—җм„ң лҸҷмһ‘н•ҳлӮҳмҡ”?

MiddlewareлҠ” мҡ”мІӯ лқјмқҙн”„мӮ¬мқҙнҒҙм—җм„ң к°ҖмһҘ лЁјм Җ мӢӨн–үлҗ©лӢҲлӢӨ. Guard, Interceptor, Pipe, Handler лӘЁл‘җ Middleware мқҙнӣ„м—җ мӢӨн–үлҗ©лӢҲлӢӨ. мқҙ л•Ңл¬ём—җ мқёмҰқ м—¬л¶ҖмҷҖ л¬ҙкҙҖн•ҳкІҢ лӘЁл“  мҡ”мІӯм—җ м Ғмҡ©лҗҳм–ҙм•ј н•ҳлҠ” лЎңк№…, мҡ”мІӯ нҢҢмӢұ, CORS мІҳлҰ¬ л“ұм—җ м Ғн•©н•©лӢҲлӢӨ. next()лҘј нҳём¶ңн•ҳм§Җ м•Ҡмңјл©ҙ мҡ”мІӯмқҙ лӢӨмқҢ лӢЁкі„лЎң м§„н–үлҗҳм§Җ м•ҠмҠөлӢҲлӢӨ.

Q useGlobalFilters()мҷҖ APP_FILTER мӨ‘ м–ҙл–Ө л°©лІ•мқҙ DIлҘј м§Җмӣҗн•ҳлӮҳмҡ”?

APP_FILTER л°©мӢқмқҙ DIлҘј м§Җмӣҗн•©лӢҲлӢӨ. main.tsмқҳ useGlobalFilters(new HttpExceptionFilter())лҠ” н•„н„°лҘј м§Ғм ‘ мқёмҠӨн„ҙмҠӨнҷ”н•ҳлҜҖлЎң NestJS DI м»Ён…Ңмқҙл„Ҳ л°–м—җм„ң мғқм„ұлҗ©лӢҲлӢӨ. л”°лқјм„ң н•„н„° лӮҙл¶Җм—җм„ң лӢӨлҘё м„ңл№„мҠӨ(ConfigService, лЎңкұ° л“ұ)лҘј мЈјмһ…л°ӣмқ„ мҲҳ м—ҶмҠөлӢҲлӢӨ. л°ҳл©ҙ AppModuleмқҳ providersм—җ { provide: APP_FILTER, useClass: HttpExceptionFilter }лҘј л“ұлЎқн•ҳл©ҙ NestJSк°Җ мқёмҠӨн„ҙмҠӨлҘј кҙҖлҰ¬н•ҳлҜҖлЎң DIк°Җ м •мғҒ мһ‘лҸҷн•©лӢҲлӢӨ.
