기본 SQL 문법은 압축하고, Postgres 고유의 핵심(JSONB · 인덱스 · MVCC · 트랜잭션 · RLS · 풀링)에 집중한다. 모든 개념은 비유 + 시각화로, 모든 SQL은 Docker + psql 순수 SQL로 직접 실행하며 배운다.

🧭 학습 방식

① 아는 영역은 MySQL ↔ PostgreSQL 비교 패널로 빠르게 전이 학습합니다(좌=MySQL, 우=PostgreSQL). ② 실습은 Docker로 postgres:18을 띄우고 psql로 순수 SQL을 직접 실행합니다 — ORM 없이 SQL 그 자체에 집중합니다. ⭐ 표시는 Postgres 고유·실무 가치가 특히 높은 레슨입니다.
